# 规范

## 格式
除了描述是自由的，日志中的其它内容应选自规范给出的范围

目前的信息格式组合方式如下

```
<对象><操作>:<描述>
```

## 对象范围（维护中）
+ 文档
+ 代码
+ 功能
+ SQL
## 操作范围（维护中）
+ 创建
+ 创建并初始化
+ 删除
+ 更新
